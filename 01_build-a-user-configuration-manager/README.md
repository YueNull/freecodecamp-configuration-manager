# Lab 01 · Build a User Configuration Manager

freeCodeCamp **Scientific Computing with Python** 认证实验。

## 功能

用 Python 字典模拟一个用户配置管理系统，支持对配置项进行增删改查：

| 函数 | 说明 |
|------|------|
| `add_setting(dict, (key, value))` | 新增配置项（已存在则拒绝） |
| `update_setting(dict, (key, value))` | 更新已有配置项（不存在则拒绝） |
| `delete_setting(dict, key)` | 删除配置项 |
| `view_settings(dict)` | 查看全部配置项 |

## 运行

```bash
python main.py
```
